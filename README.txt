* How to use *

-> Go at https://kirchner.center/gmd/#/
-> Select a category on the top right corner to request on (default : diseases)
-> Type your search in the search field (it must match exactly)
-> The results are displayed and grouped by category (diseases, drugs, side effects, symptoms)
